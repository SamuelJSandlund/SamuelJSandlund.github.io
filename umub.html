UMU-Book

During a course in databases me and a group of classmates set out to investigate the popular field of social media and how relational databases can be used to provide such a service. The result was a twitter-like platform that we call UMU-Book.

The platform was built using .NET with a Microsoft SQL-server database. To get a deeper understanding of the database side of things we opted to write our own data access layer instead of using Entity Framework.

My parts in the project

  • Content algorithm for the user feed
  • Database architecture
  • Post interactions
